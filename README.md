# SeekNote

> ## 这是一个关于Thinkphp5框架搭建应用
>>### 主要功能：
>><ol>
>> <li>编辑-->（富文本）==完成</li>
>> <li>修改-->（单条修改-列表修改）==完成</li>
>> <li>显示-->（分类显示）==完成</li>
>> <li>查找-->（多维度查找）==进行中</li>
>> <li>留言==完成</li>
>></ol>
>
>
>>### 前期准备
>><ol>
>> <li>搭建本地虚拟域名配置</li>
>> <li>下载/clone代码至虚拟目录下</li>
>> <li>虚拟域名配置路径指向所下载的文件中/public目录</li>
>> <li>将Git代码中的根目录中的sql语句导入数据库（数据库名为seek）</li>
>> <li>网页打开所配置的虚拟域名即可预览</li>
>></ol>
